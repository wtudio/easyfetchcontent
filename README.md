# easyfetchcontent

This project provides some cmake modules which can be used to import some famous libraries to your own project based on cmake fetchcontent.

## How to use?
Copy the **xxlib/GetXXlib.cmake** files to your `CMAKE_MODULE_PATH` and include them. Refer to the demo provided in the same folder.

## Supported Libraries
- [asio](asio)
- [fmt](fmt)
- [freeglut](freeglut)
- [gflags](gflags)
- [googlebenchmark](googlebenchmark)
- [gtest](gtest)
- [jsoncpp](jsoncpp)
- [leveldb](leveldb)
- [libevent](libevent)
- [libtomcrypt](libtomcrypt)
- [lua](lua)
- [protobuf](protobuf)
- [rapidjson](rapidjson)
- [robinhood](robinhood)
- [snappy](snappy)
- [spdlog](spdlog)
- [sqlite](sqlite)
- [tinyxml2](tinyxml2)
- [yamlcpp](yamlcpp)
